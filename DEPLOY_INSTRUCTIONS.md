## 📦 Cara Deploy ke Vercel (Pilih Salah Satu):

### **CARA 1: Via Vercel Dashboard (TERCEPAT - 2 menit)**

1. Buka: **https://vercel.com/new**
2. Login dengan GitHub
3. **Import Repository:**
   - Cari repository: `gryowaldo13-ai/SHARELOck`
   - Klik "Import"
4. **Deploy Settings:**
   - Project Name: `sharelock` (atau nama lain)
   - Framework Preset: Other
   - Root Directory: `./`
   - Build Command: (kosongkan)
   - Output Directory: (kosongkan)
5. Klik **"Deploy"**
6. ✅ **SELESAI!** Tunggu 1-2 menit

URL production akan muncul seperti: `https://sharelock-xxx.vercel.app`

---

### **CARA 2: Via Command Line**

Buka Command Prompt di folder ini, lalu:

```cmd
npx vercel --prod
```

Atau double-click file: **`deploy.bat`**

Ikuti instruksi:
- Login (browser otomatis terbuka)
- Link to existing project? **N** (No)
- Project name: **sharelock**
- Directory: **./​** (tekan Enter)
- Override settings? **N** (No)

✅ Deploy selesai!

---

### **CARA 3: Auto Deploy (Setup Sekali)**

Setelah connect ke Vercel (cara 1 atau 2):
- Setiap `git push` otomatis deploy
- Tidak perlu manual deploy lagi
- Vercel akan monitoring repository
